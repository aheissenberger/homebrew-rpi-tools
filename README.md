# Aheissenberger Rpi-tools

## How do I install these formulae?

`brew install aheissenberger/rpi-tools/<formula>`

Or `brew tap aheissenberger/rpi-tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aheissenberger/rpi-tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
